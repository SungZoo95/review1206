UX = user experience (과정) - 의도와 기획 
- 카카오톡 친구추가를 아이디로도 할 수 없을까? -UX

UI = user interface (사용자가 편하게 만들어주는) - 결과 
- 카카오톡 아이디로 추가 할 수 있게 만들었다 - UI

어포던스 : 행동 지원성 
- 내가 뭔가를 하지 않아도 감각적으로 알 수 있게 만들어준다.
억셉티드 : 환경건축
- 공간연출 
유럽 그린포비아 : ~스탄 , 몽골족 , 훈족  , 페르시아(상징 : 초록색)
구독경제 : 정액제를 통해 지속적으로 서비스를 받는 경제 활동
- 무인자동차를 통해 컨텐츠 개발 
잊지 말아야 할 UX_UI의 세가지 요소
- 1. 사용성과 기능성 
	- 사용자가 중심이 되어야 한다. 
	- 물자가 귀해서 선택을 강요 받던 시대에서는 기능성에
	  중심을 주었지만, 지금은 선택을 하는 사용자의 사용성이
	  중요하다
- 2. 피드백 
	- 바로바로 오는 피드백(즉각적인)의 중요성 (댓글 반응, 고전 컨텐츠를
	  살펴볼 필요성이있다.
- 3. 인간적인 감성
	- 다양한 리서치로 알아보기 
	- 인간적인 감성에 어필해야한다. 
	- 정보를 취합하는 능력이 중요하다.
	- 의미를 캐내는 과정은 스토리, 스토리텔링이 중요하다.
